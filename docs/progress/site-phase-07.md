## Outcome

The release-story system is complete. Published release notes normalize into
an archive, detail pages, categories, dates, verification/source/download
links, RSS, upgrade warnings, and the homepage module. The current archive is
truthfully empty because GitHub exposes no release. The roadmap separates
implemented, committed, and exploratory work without a date promise.

`/.github/RELEASE_TEMPLATE.md` and `/.github/release.yml` define future
release narrative and categories.
